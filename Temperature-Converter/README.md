# 🌡️ Temperature Converter Web Application

## 📌 Objective
To develop a simple web application that converts temperature values between Celsius, Fahrenheit, and Kelvin.

---

## 🛠️ Tools Used
- HTML5
- CSS3
- JavaScript
- VS Code

---

## 🪜 Steps Performed
1. Created input fields for temperature entry
2. Added dropdown for unit selection
3. Implemented conversion logic using JavaScript
4. Displayed real-time output
5. Tested accuracy of conversions

---

## 🎯 Outcome
A functional temperature converter application that performs accurate and instant unit conversions.
